# People Groups Admin Page

## Overview

The people groups admin page is where you view and manage all people groups in the system. It follows the same list-and-detail layout used across the admin area — a searchable list on the left, with a detail panel that slides open on the right when you select a group.

## The List

Each people group in the list shows:

- **Image** — The group's photo (if one has been uploaded)
- **Name**
- **Badges** — An "Adopted" badge if any group has adopted this people group, and an engagement status badge
- **Population** — The group's population count
- **Prayer stats** — Four metrics: people committed, time pledged, people praying, and daily prayer time

You can search by name to find a specific group. The total count of people groups is shown below the search field.

## The Detail Panel

Clicking a people group opens its detail panel with the following sections:

### Quick actions

Buttons at the top provide shortcuts to:

- **Subscribers** — Jump to the subscribers page filtered to this people group
- **Content** — Go to the content management page for this group's libraries
- **Open** — Open the public-facing people group page in a new tab

### Adopted by

Shows which groups (churches/organizations) have adopted this people group. Each adoption is shown as a card. You can add a new adopting group by clicking "+ Add" and selecting from a dropdown.

### Editable fields

The remaining fields are organized by category and vary based on the group's data. These include basic information (name, image, Joshua Project ID), descriptions (which support multiple languages via translatable fields), and various metadata fields. Some fields are read-only.

Changes are saved by clicking the "Save" button in the panel header.

### Activity and comments

Two tabs on the right side of the panel show:

- **Activity** — A log of all changes made to this record, showing who changed what and when
- **Comments** — Team notes and discussion about this people group (see [Comments & Mentions](comments-and-mentions.md))

## Current Limitations

- Cannot create or delete people groups from this page — they are imported
- No bulk editing of multiple groups at once from the list view
