# Comments & Mentions

## Overview

Comments let admins leave notes and have conversations on records in the system — people groups, subscribers, and groups. They support rich text formatting and @mentions that notify other team members by email.

## Where Comments Appear

Comments are available on the detail panel of any record type in the admin area. When you open a people group, subscriber, or group, you'll find a Comments section where you can view and add comments.

Comments are shown newest first, with the author's name and a relative timestamp (e.g., "3h ago"). System-generated comments (such as automated notes from record changes) are visually distinguished with a "System" badge.

## Writing a Comment

The comment editor supports basic rich text formatting:

- Bold, italic, and strikethrough
- Bullet and numbered lists
- Links

Click "Add Comment" to post. You can edit or delete your own comments after posting — a pencil icon opens inline editing, and a trash icon removes the comment after confirmation.

## Mentioning Team Members

Type `@` in the comment editor to see a dropdown of admin users. Start typing a name to filter the list, then click or press Enter to insert the mention. Mentions appear highlighted in the comment.

When you mention someone, they receive an email notification showing who mentioned them, which record the comment is on, and the comment text, with a link to view it directly.

You won't be notified about your own mentions.

## When Records Are Deleted

If a record is deleted, all comments attached to it are automatically removed as well.

## Current Limitations

- Only admin users can be mentioned — subscribers and external contacts cannot
- No way to subscribe to comments on a record without being mentioned
- No in-app notifications — mentions only send email
- Cannot attach files to comments
