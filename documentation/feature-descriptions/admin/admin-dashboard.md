# Admin Dashboard

## Overview

The admin dashboard is the landing page when you log into the admin area. It provides quick navigation to all major sections and an at-a-glance view of platform health through statistics and charts.

## Quick Navigation

The top of the dashboard shows links to the main admin sections: People Groups, Subscribers, Groups, Libraries, Users, and Profile. Each link has an icon for quick identification.

## Statistics

The dashboard has two tabs: **General** and **Prayer**.

### General Tab

**Three overview charts** show high-level coverage:

- **People Groups with Prayer** — how many people groups have active prayer campaigns versus those without
- **People Group Adoption** — how many groups have been adopted versus unadopted
- **People Group Engagement** — how many groups are actively engaged versus unengaged

**Four metric cards** show prayer time numbers:

- **Total Prayer Time Committed** — the sum of all active subscribers' pledged prayer durations
- **Daily Prayer Time Committed** — committed time expressed as a daily average
- **Total Prayer Time Recorded** — actual prayer time logged across all groups
- **Last 24h Prayer Time Recorded** — recent prayer activity

### Prayer Tab

Three charts show trends over the **last 30 days**:

- **Unique People Praying** — a daily bar chart showing how many signed-up subscribers versus how many actually prayed each day
- **Daily Prayer Recorded** — minutes of prayer activity logged each day
- **Daily Prayer Committed** — committed prayer time by subscription each day

Hovering over any bar shows the exact value and date.

## Current Limitations

- Stats are read-only — no drill-down into individual people groups from the charts
- Prayer charts are fixed to a 30-day window
- No way to export dashboard data
